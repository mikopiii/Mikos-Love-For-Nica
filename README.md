## Mobile Deployment Guide

1. **GitHub**: Create a repo and copy-paste the code from all files into new files on the GitHub website.
2. **Netlify**: Connect your GitHub repo.
3. **API Key**: Add your Gemini API key to Netlify's Environment Variables as `API_KEY`.
4. **Build Settings**: 
   - Build Command: `npm run build`
   - Publish Directory: `dist`

Built with React, Vite, Tailwind CSS, and Google Gemini API.
